có thể gộp các padding (top,bottom,left,right) vào 1 thuộc tính duy nhất là padding
    ví dụ: padding:20px;

Quy tắc viết tắt của padding thì giống hệt margin
TH1 : padding: 30px; => padding-top: 30px; padding-bottom: 30px; padding-left: 30px; padding-right: 30px;
TH2 : padding: 30px 10px 6px 20px; => padding-top: 30px; padding-bottom: 20px; padding-left: 6px; padding-right: 10px;
TH3 : padding: 5px 9px 12px; => padding-top: 5px; padding-bottom: 9px; padding-left: 9px; padding-right: 12px;
TH4 : padding: 10px 20px; => padding-top: 10px; padding-bottom: 0; padding-left: 0; padding-right: 20px;
DUY : óc bã đậu
DŨNG : óc lìn
